# LLM Fine-Tuning Project

This project demonstrates how to fine-tune an open-source Large Language Model (LLM) using custom datasets and run inference on the trained model.

---

# Model = Qwen-2.5-7B-Instruct
# Data = https://huggingface.co/datasets/microsoft/orca-math-word-problems-200k

# Features

- Fine-tuning open-source LLMs
- Jupyter Notebook workflow
- Hugging Face integration
- Dataset preprocessing
- Training and inference pipeline
- GPU acceleration support
- Compatible with Google Colab and local systems

---

# Tech Stack

- Python
- PyTorch
- Transformers
- Hugging Face Datasets
- Unsloth
- Jupyter Notebook
